# Model 2 decision tables

This step condenses the application-scaling grid into product-facing tables:

- minimum active membrane area for selected desired air flows,
- minimum flux for selected desired air flows,
- best latent fraction/regime at each feasible point,
- search-ceiling flags where the air-flow grid was too small.

Run it after `run_model2_load_area_scaling.py`, preferably with a wide air-flow range.
